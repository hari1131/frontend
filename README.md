# 🛍️ Appu's Accessories

> *"Complete Your Tech Life"*

A simple multi-page static website for a mobile accessories shop built using plain HTML and CSS.

---

## 📁 Project Structure

```
appu-accessories/
│
├── product.html       # Home Page
├── product2.html      # About Page
├── product3.html      # Products Page
├── product4.html      # Contact Page
│
├── about.jpg          # About page image
├── 1.jpeg – 12.jpeg   # Product images
└── WhatsApp Image ... # Hero/background image
```

---

## 📄 Pages Overview

### 🏠 Home (`product.html`)
- Navigation bar with links to all pages
- Background image with shop logo and tagline
- Displays the brand name **APPU'S ACCESSORIES**

### ℹ️ About (`product2.html`)
- Information about the shop
- Highlights quality, affordability, and fast delivery
- Features an about image

### 🛒 Products (`product3.html`)
- Grid layout showing **12 accessories**
- Each product card has: image, name, price, and "Order Now" button
- Products include: Watch, Headset, AirPods, Back Cover, Cables, Pendrive, SSD, Speakers, Tripod, Camera Lens, Charger, Power Bank

### 📞 Contact (`product4.html`)
- Contact form (Name, Email, Message)
- Shop address, phone number, email, and working hours

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Page structure |
| CSS3 | Styling and layout |
| CSS Grid | Product card layout |

> No frameworks or JavaScript libraries used — pure HTML & CSS!

---

## 🚀 How to Run

1. Clone or download this repository
2. Make sure all image files are in the **same folder** as the HTML files
3. Open `product.html` in any browser

```bash
git clone https://github.com/your-username/appu-accessories.git
cd appu-accessories
# Open product.html in your browser
```
